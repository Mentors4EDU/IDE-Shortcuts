1. Open File: ``Ctrl+P`` 
2. [Find and replace](https://www.gitpod.io/docs/search/): ``Ctrl+Shift+F``
3. Open Workspace: ``Ctrl+O``
4. Command Palette: ``F1``
5. Format Document: ``Alt+Shift+F``
6. Next Tab: ``Alt+Page Down``
7. Pevious Tab: ``Alt+Page Up``
